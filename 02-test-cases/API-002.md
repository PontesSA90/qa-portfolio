# API-002 — Criar post SEM título via API pública JSONPlaceholder (POST /posts)

- **Sistema sob teste:** API pública JSONPlaceholder (`https://jsonplaceholder.typicode.com`)
- **Contexto:** Exercício prático para portfólio de QA de API (não relacionado ao produto AbacusAI).
- **Objetivo:** Avaliar o comportamento da API quando um campo possivelmente obrigatório (`title`) é omitido no corpo da requisição.
- **Endpoint:** `POST https://jsonplaceholder.typicode.com/posts`
- **Ferramenta:** Postman
- **Pré-condição:** Postman instalado e acesso à internet.

## Dados de entrada (request body)

```json
{
  "body": "Conteúdo de teste sem título para validar comportamento da API.",
  "userId": 1
}
