# TC-004 — Atalhos Enter e Shift+Enter no campo de mensagem

- **Objetivo:** Verificar o comportamento das teclas Enter e Shift+Enter no campo de entrada de texto do chat.
- **Ambiente:** Windows 11 / Aplicativo AbacusAI (desktop)
- **Pré-condição:** Usuário logado; conversa aberta.

## Passos
1. Abrir o aplicativo AbacusAI e uma conversa existente.
2. Digitar um texto curto no campo de mensagem (ex.: "Olá").
3. Pressionar a tecla **Enter**.
4. Observar o comportamento.
5. Digitar outro texto curto no campo de mensagem (ex.: "Teste").
6. Pressionar a combinação de teclas **Shift + Enter**.
7. Observar o comportamento.
8. Inserir múltiplas linhas vazias usando Shift+Enter.
9. Digitar mais texto.
10. Enviar a mensagem.

## Resultado esperado
- Ao pressionar **Enter**, a mensagem deve ser enviada.
- Ao pressionar **Shift + Enter**, uma nova linha deve ser inserida no campo de texto.
- Múltiplas linhas vazias inseridas com Shift+Enter devem ser preservadas na mensagem enviada, mantendo a formatação visual.

## Resultado obtido
- Enter envia a mensagem.
- Shift+Enter insere nova linha.
- Múltiplas linhas vazias são ignoradas no envio final (o que foi documentado como melhoria no BR-004).

## Status
**Falhou** (devido à remoção de linhas vazias, conforme BR-004)

## Observações
Este caso de teste falha devido ao comportamento de remoção de linhas vazias, que foi documentado como uma melhoria de usabilidade no BR-004. O comportamento de Enter e Shift+Enter para enviar e inserir nova linha, respectivamente, está conforme o esperado.
